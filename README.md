# StudyProgressTracker (Python + SQLite)

Een simpele Python console-app om studievoortgang per vak bij te houden.

## Functies
- Vakken toevoegen
- Studie-uren registreren
- Overzicht van totaal aantal uren per vak
- Opslag in SQLite-database

## Starten
1. Open de map in Visual Studio Code
2. Open een terminal
3. Run:
   python study_progress.py
